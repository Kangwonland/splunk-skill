# SPL Commands by Category

| Category | Commands |
|----------|----------|
| **Correlation** | `associate`, `correlate`, `contingency` |
| **Data and Indexes** | `dbinspect`, `eventcount`, `inputlookup`, `loadjob`, `metadata`, `metasearch`, `rest`, `savedsearch` |
| **Fields — Add/Extract** | `erex`, `extract`/`kv`, `kvform`, `multikv`, `rex`, `spath`, `xmlkv`, `xpath` |
| **Fields — Modify** | `convert`, `eval`, `fieldformat`, `filldown`, `fillnull`, `makemv`, `nomv`, `rename`, `replace`, `strcat` |
| **Find Anomalies** | `analyzefields`, `anomalies`, `anomalydetection`, `anomalousvalue`, `cluster`, `kmeans`, `outlier` |
| **Geographic** | `geom`, `geomfilter`, `geostats`, `iplocation` |
| **Metrics** | `mcollect`, `mpreview`, `mstats` |
| **Prediction** | `predict`, `trendline`, `x11` |
| **Reports** | `addcoltotals`, `addtotals`, `bin`/`bucket`, `chart`, `delta`, `eventstats`, `gauge`, `pivot`, `rangemap`, `reltime`, `sort`, `stats`, `streamstats`, `timechart`, `timewrap`, `top`, `rare`, `tstats` |
| **Results — Append/Combine** | `append`, `appendcols`, `appendpipe`, `collect`, `join`, `lookup`, `outputcsv`, `outputlookup`, `sendemail`, `set` |
| **Results — Filter** | `dedup`, `fields`, `head`, `regex`, `search`, `tail`, `uniq`, `where` |
| **Results — Format** | `highlight`, `iconify`, `table`, `transpose`, `untable`, `xyseries` |
| **Results — Generate** | `gentimes`, `inputcsv`, `makeresults` |
| **Results — Group** | `mvcombine`, `mvexpand`, `selfjoin`, `transaction` |
| **Results — Reorder** | `reverse`, `rtorder` |
| **Subsearch** | `append`, `format`, `return` |
| **Time** | `bin`, `gentimes`, `makecontinuous`, `reltime` |

Source: https://help.splunk.com/en/splunk-enterprise/search/spl-search-reference/9.4
